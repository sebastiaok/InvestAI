# AI 투자분석 Multi-Agent 프로젝트 기획서

## 1. 과제 주제
나만의 End-to-End AI Agent 서비스 개발: AI 투자분석 Multi-Agent

## 2. 문제 정의
투자자는 매일 뉴스, 가격, 재무, 포트폴리오, 리스크를 따로 확인해야 한다.
정보 소스가 분산되어 있고 해석 기준도 제각각이라 빠르고 일관된 분석이 어렵다.

## 3. 해결 방향
사용자 질문을 Planner가 해석하고, 전문 Agent들이 역할 분담하여 데이터를 모으고,
RAG 기반 내부 리서치 노트까지 결합해 최종 투자 브리프를 생성한다.

## 4. 주요 사용자
- 개인 투자자
- 리서치 보조가 필요한 주니어 애널리스트
- 포트폴리오 현황을 빠르게 점검하려는 실무자

## 5. 차별화 포인트
- 단순 종목 요약이 아니라 A2A 기반 협업 구조
- 긍정 시나리오와 부정 시나리오를 함께 제공
- 포트폴리오 관점 영향 분석
- 근거와 리스크를 분리해 설명하는 실무형 출력

## 6. MVP 범위
- 종목명 또는 질문 입력
- 뉴스/시장/기초재무/포트폴리오 입력 분석
- RAG 기반 내부 메모 검색
- 종합 리포트 생성

## 7. 확장 범위
- 브로커 API 연결
- 종목 비교 모드
- 섹터 로테이션 분석
- 알림 자동화
